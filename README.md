# Sentidos Cobranças

Este repositório contém o sistema de cobrança automatizada do Instituto Sentidos e FAEPI.

## Como Rodar Localmente

**Pré-requisitos:** Node.js v18+

1. Instale as dependências:
   `npm install`
2. Inicie o servidor backend (API):
   `npm run server`
3. Inicie o servidor frontend (Vite):
   `npm run dev`
4. Acesse no navegador:
   `http://localhost:3000`
